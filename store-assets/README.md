## 🚀 How to Use These Assets

### Apple App Store
1. Log in to App Store Connect
2. Go to your app → App Store tab
3. Under "App Previews and Screenshots"
4. Upload screenshots from `/app/store-assets/screenshots/ios/`
5. Add captions using descriptions above

### Google Play Store
1. Log in to Play Console
2. Go to Store presence → Main store listing
3. **Feature Graphic:** Upload `feature-graphic.png` (REQUIRED)
4. **Screenshots:** Upload from `/app/store-assets/screenshots/android/`
5. Add descriptions
